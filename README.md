##### Instalar FLEX Linux/Ubuntu:
```
    $ sudo apt-get update
    
    $ sudo apt-get install flex

```

##### Instalar BISON Linux/Ubuntu:
```
    $ sudo apt-get update
    
    $ sudo apt-get install bison

```

```
    $ flex analisador_lex_Cminus_base.l
    
    $ gcc lex.yy.c -o saida
    
    $ ./saida
```
